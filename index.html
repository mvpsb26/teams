🛠 Script Setup Guide
Deploy the backend to Google Apps Script to power this app.
  • Go to script.google.com → New Project.
  • Delete existing code, paste the script (button below), save.
  • Deploy → New Deployment → Web app → Execute as: Me → Access: Anyone → Deploy.
  • Authorize when prompted (needs Gmail + Drive).
  • Copy the Web App URL → paste in settings above.
